## Siriraj Single Sign-On (SSO)

บัญชีสำหรับใช้งานระบบของศิริราช ตัวอย่างที่ใช้บ่อย เช่น Canvas, Selecx, SICMS, Wi-Fi คณะฯ, Si vWORK

สำหรับนักศึกษาใหม่ สามารถเปิดใช้งานบัญชีได้โดยการรีเซ็ตรหัสผ่านใน
[E-Service](https://si-eservice3.si.mahidol.ac.th/selfservice/)

### ชื่อผู้ใช้ (Username)

<ParamField path="firstname.sur">ชื่อจริง + `.` + นามสกุล 3 ตัวแรก (ภาษาอังกฤษ)</ParamField>

### เปลี่ยนรหัสผ่าน

รหัสผ่านจะหมดอายุทุก 90 วัน ควรเปลี่ยนรหัสผ่านก่อนสอบเพื่อป้องกันปัญหาการเข้าระบบ

## Mahidol Account

บัญชีสำหรับใช้งานระบบของ ม.มหิดล ตัวอย่างที่ใช้บ่อย เช่น MU-WiFi, Google Workspace (.edu), Microsoft 365 (.ac.th)

สำหรับนักศึกษาใหม่
สามารถเปิดใช้งานบัญชีได้[ที่นี่](https://myinternet.mahidol.ac.th/student/activate)
ดูวิธีเปิดใช้งานได้[ที่นี่](https://muit.mahidol.ac.th/docs/users-guide/manual-it-membership-account/manual-it-member-acc-reg/)

### ชื่อผู้ใช้ (Username)

มีทั้งหมด 3 รูปแบบ ขึ้นกับแพลตฟอร์มที่เข้าใช้งาน ดูได้จากตารางในหัวข้อถัดไป ทุกแบบ**ใช้รหัสผ่านเดียวกัน**

<ParamField path="uYY01XXX">
  YY = ปี พ.ศ. ที่เข้าเรียน, 01 = รหัสคณะศิริราชฯ, XXX = รหัสนักศึกษา สำหรับ Mahidol Authen
  ซึ่ง**ใช้เข้าบริการส่วนใหญ่**
</ParamField>
<ParamField path="firstname.sur@student.mahidol.ac.th">
  ต่อไปจะเรียกว่า **.edu** สำหรับ SI-WiFi, MU-WiFi, Microsoft 365, Adobe Creative Cloud และใช้แทน
  `uYY01XXX` ได้ในบางบริการ
</ParamField>
<ParamField path="firstname.sur@student.mahidol.edu">
  ต่อไปจะเรียกว่า **.ac.th** สำหรับ Google Workspace และบริการอื่นที่ใช้ Google Account
  ในการเข้าใช้งาน
</ParamField>
