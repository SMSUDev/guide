## Title and Description

ชื่อเรื่อง (Title) จะถูกใช้แสดงในเมนูด้วย สามารถตั้งได้จากส่วนหัวของหน้าเนื้อหาใน Web Editor

<Info>คำอธิบาย (Description) ไม่จำเป็นต้องตั้งก็ได้ หากไม่ตั้งก็จะไม่แสดงผลในหน้าเนื้อหาจริง</Info>

![ตั้งชื่อเรื่องและคำอธิบาย](/images/it/guide/managepages/1-title.webp)

## Sidebar Title

หากชื่อเรื่องยาวเกินไป สามารถตั้งชื่อที่แสดงในเมนู (Sidebar Title) ด้วยชื่อที่สั้นลงได้

<Steps>
  <Step title="กดที่ปุ่มไอคอนตั้งค่า ด้านขวาของชื่อเรื่อง">
    ![ตั้งชื่อที่แสดงในเมนู](/images/it/guide/managepages/2-pagesetting.webp)
  </Step>
  <Step title='ที่หัวข้อ "Sidebar Title" พิมพ์ชื่อสั้นในช่อง "Short Title"'>
    ![พิมพ์ชื่อสั้น](/images/it/guide/managepages/3-addshortname.webp)
  </Step>
</Steps>

## Sidebar Icon

หากต้องการให้หน้าเนื้อหานั้นมีไอคอนแสดงในแถบเมนู สามารถตั้งค่าได้ที่ปุ่มข้างช่อง "Short Title" ที่ใช้พิมพ์ชื่อสั้นในหัวข้อด้านบน แล้วพิมพ์ค้นหาไอคอนที่ต้องการ

<Tip>
  ไอคอนที่ใช้ในนี้มาจาก Font Awesome สามารถค้นหาไอคอนจาก [เว็บไซต์ Font
  Awesome](https://fontawesome.com/icons) ได้โดยตรง (สามารถค้นหาได้ง่ายกว่า)
  แล้วกดที่ชื่อไอคอนเพื่อคัดลอกมาใส่ในช่องค้นหาของ Web Editor

  <u>หมายเหตุ</u> สามารถเลือกได้ทั้งไอคอนฟรี และไอคอนที่ติดแท็ก "Pro"
</Tip>

![ตั้งค่าไอคอนที่แสดงในแถบเมนู](/images/it/guide/managepages/4-addicon.webp)
