## ข้อควรรู้เบื้องต้น

Guide นี้ใช้ **Mintlify** เป็น documentation platform ซึ่งมี Web Editor ให้สามารถแก้ไขได้แบบ What-You-See-Is-What-You-Get (WYSIWYG)

## เริ่มต้นแก้ไข

<Steps>
  <Step title="เข้าใช้งาน Web Editor">
    เข้าใช้งานที่ [dashboard.mintlify.com](https://dashboard.mintlify.com) ด้วยบัญชี Google
    ที่ได้รับจากฝ่าย IT
    <Tip>
      **แนะนำให้เข้าใช้งานผ่านคอมพิวเตอร์** หรืออุปกรณ์ที่มีหน้าจอขนาดใหญ่
      เนื่องจากการใช้งานบนโทรศัพท์มือถืออาจมีข้อจำกัดในการแสดงผลและการใช้งานเมนู
    </Tip>
  </Step>
  <Step title={`เลือกเมนู "Editor"`}>
    จากแถบด้านข้างบนคอมพิวเตอร์ หรือปุ่มเมนูมุมบนซ้ายบนโทรศัพท์มือถือ ![เลือกเมนู Editor
    จากแถบด้านข้าง](/images/it/guide/intro/0-editor.webp)
  </Step>
  <Step title="เลือกไฟล์ที่ต้องการแก้ไข">
    วิธีการจัดการไฟล์จะอธิบายในหน้าต่อ ๆ ไป [ดูวิธีเลือกไฟล์/สร้างไฟล์ใหม่](/it/guide/edit/file)
    ![เลือกไฟล์ที่ต้องการแก้ไข](/images/it/guide/intro/1-selectfile.webp)
  </Step>
  <Step title="เริ่มแก้ไขเนื้อหา">![หน้าต่าง Editor](/images/it/guide/intro/2-startedit.webp)</Step>
  <Step title="เพิ่ม component สำเร็จรูป">
    พิมพ์ `/` เพื่อแสดงรายการ component สำเร็จรูปที่สามารถเพิ่มได้ ![เพิ่ม component
    สำเร็จรูป](/images/it/guide/intro/3-compmenu.webp)
  </Step>
</Steps>
