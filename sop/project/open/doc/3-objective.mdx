เป็นส่วนที่อยากให้ทุกคนให้ความสำคัญ เพราะเป็น**เครื่องชี้แนวทางในการดำเนินโครงการ**ที่ทีมงานมีร่วมกัน **ไม่จำเป็นต้องซ้ำกับปีก่อน ๆ** แต่ขอให้เป็นวัตถุประสงค์ของโครงการนี้จริง ๆ และ**ทีมงานได้มีส่วนร่วมและรับรู้วัตถุประสงค์ร่วมกัน** และไม่มีความซ้ำซ้อนระหว่างข้อ สามารถมีกี่ข้อก็ได้ แต่**ไม่ควรมีเกิน 3 ข้อ** เพื่อให้เราสามารถโฟกัสกับสิ่งที่เราต้องการได้จริง อาจยึดตามหลัก SMART Goals

## SMART Goals

- **S = Sensible** (เป็นไปได้) หมายถึง วัตถุประสงค์จะต้องมีความเป็นไปได้ ในการดำเนินงานโครงการ
- **M = Measurable** (วัดได้) หมายถึง วัตถุประสงค์ที่ดีจะต้องสามารถวัดและประเมินผลได้
- **A = Attainable** (ระบุสิ่งที่ต้องการ) หมายถึง วัตถุประสงค์ที่ดีต้องระบุสิ่งที่ต้องการดำเนินงานอย่างชัดเจน และเฉพาะเจาะจงมากที่สุด
- **R = Resonable** (เป็นเหตุเป็นผล) หมายถึง วัตถุประสงค์ที่ดีต้องมีความเป็นเหตุเป็นผลในการปฏิบัติ
- **T = Time** (เวลา) หมายถึง วัตถุประสงค์ที่ดีจะต้องมีขอบเขตของเวลาที่แน่นอนในการปฏิบัติงาน
