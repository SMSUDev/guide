## ตราและที่อยู่สโมสรนักศึกษาแพทย์ศิริราช

ใช้รูปแบบล่าสุดตามรูปด้านล่าง หรือตามเทมเพลต

![ตราและที่อยู่สโมสรนักศึกษาแพทย์ศิริราช](/images/sop/proj/project_open_header_logo.png)

<Warning>ระวังใบโครงการบางฉบับยังเป็นรูปแบบเก่า ให้แก้ไขเป็นรูปแบบนี้</Warning>

ใส่ชื่อโครงการเต็ม ถ้ามีครั้งที่หรือปีให้ใส่มาด้วย พยายามใช้ภาษาทางการ โดย**ไม่ต้องใส่ Theme ของงาน**

![ชื่อโครงการและวันที่จัดงาน](/images/sop/proj/project_open_header_date.png)

## วัน/ช่วงเวลาที่จัดงาน
