## ส่วนแรก เขียนตามวัตถุประสงค์

![คำแนะนำการเขียนส่วนแรก](/images/sop/proj/project_open_goal_part1.webp)

นำ**วัตถุประสงค์**โครงการ ใน[ใบเปิดโครงการข้อ 3. วัตถุประสงค์](/sop/project/open/doc/3-objective) มาเขียน

- **ตัวชี้วัดโครงการ** เขียนสิ่งที่สะท้อนถึงความสำเร็จของโครงการตามวัตถุประสงค์ และ<u>สามารถวัดผลได้จริง</u>
- **เป้าหมาย** พิจารณาจากข้อมูลจากปีที่แล้ว, ความเป็นไปได้ หรือ Literature review
- **วิธีการประเมินผล** วิธีการประเมินสอดคล้องกับชนิดของข้อมูลที่เก็บตามตัวชี้วัด (พร้อมระบุช่วงระดับหรือคะแนนเต็ม) ดูตัวอย่างได้จากตารางด้านล่าง

### ตัวอย่างวิธีการประเมินผล

| ข้อมูล                          | วิธีการประเมินผลและเครื่องมือ                       | แหล่งข้อมูล                       | เวลา                           |
| ------------------------------- | --------------------------------------------------- | --------------------------------- | ------------------------------ |
| ข้อมูลเชิงปริมาณ                | Document review                                     | Depend                            | หลังจบกิจกรรม, หลังจากมีข้อมูล |
| ความพึงพอใจ / ความรู้สึก        | Survey via questionnaire                            | ผู้เข้าร่วม                       | หลังจบกิจกรรม                  |
| ความรู้ ความเข้าใจ ที่เพิ่มขึ้น | Pre-post test (Self assessment, Quiz, Open ended q) | ผู้เข้าร่วม                       | หลังจบกิจกรรมทันที/ระยะหนึ่ง   |
| ทักษะที่เพิ่มขึ้น               | Pre-post test (Self assessment, Observation)        | ผู้เข้าร่วม, วิทยากร, Facilitator | หลังจบกิจกรรมทันที/ระยะหนึ่ง   |
| การเปลี่ยนแปลงทัศนคติ           | Survey, Focus group, Reflection                     | ผู้เข้าร่วม                       | หลังจบกิจกรรมระยะหนึ่ง         |
| การเปลี่ยนแปลงพฤติกรรม          | Survey, Reflection, Interview                       | ผู้เข้าร่วม                       | หลังจบกิจกรรมระยะหนึ่ง         |
| ประสิทธิภาพการจัดงาน            | สรุปการทำงาน, Feedback ภายใน, ประชุม reflection     | ผู้จัดกิจกรรม                     | หลังจบกิจกรรม                  |

## ส่วนที่สอง ตัวชี้วัดที่จำเป็นต้องเก็บ

![คำแนะนำการเขียนส่วนที่สอง](/images/sop/proj/project_open_goal_part2.webp)

**บังคับทุกโครงการเก็บและสรุปข้อมูล** ทั้งหมด 5 ข้อ

1. ร้อยละของนักศึกษาแพทย์ที่เห็นว่ากิจกรรมตรงตามความต้องการ
2. ร้อยละของผู้เข้าร่วมที่มั่นใจในการแนะนำกิจกรรมนี้ให้ผู้อื่นเข้าร่วมระดับ 4 - 5 (จาก scale 0 - 5)
3. ร้อยละของผู้เข้าร่วมที่ประเมินว่ากิจกรรมช่วยพัฒนาทักษะ…_ทักษะที่ได้จากการเข้าร่วมกับกิจกรรม_… อยู่ในระดับ 3 (จาก scale 0 - 3)
4. ค่าเฉลี่ยของระดับความพึงพอใจในภาพรวม (จาก scale 0 - 5)
5. ร้อยละของความพึงพอใจระดับ 3 - 5 (จาก scale 0 - 5)

## เกณฑ์จำนวนผู้ตอบแบบสอบถาม

![คำแนะนำการเขียนเกณฑ์จำนวนผู้ตอบแบบสอบถาม](/images/sop/proj/project_open_goal_footer.webp)

คำนวณจำนวนคนจาก [ใบเปิดโครงการข้อ 4.](/sop/project/open/doc/4-participant) และใช้เกณฑ์ต่อไปนี้

- **ผู้จัดโครงการ**ตั้งแต่ <u>ร้อยละ 70</u> ขึ้นไปของผู้จัดโครงการทั้งหมด
- **ผู้เข้าร่วมโครงการ**มากกว่าเท่ากับร้อยละ ดังเงื่อนไขต่อไปนี้
  - หากจำนวนมากกว่า 300 คน ต้องมากกว่าเท่ากับ <u>ร้อยละ 40</u> ของจำนวนผู้เข้าร่วมโครงการทั้งหมด
  - หากจำนวนน้อยกว่า 300 คน ต้องมากกว่าเท่ากับ <u>ร้อยละ 60</u> ของจำนวนผู้เข้าร่วมโครงการทั้งหมด
