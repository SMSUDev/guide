1. คำว่า **”รายงานสรุปโครงการ”**
2. **ชื่อโครงการ** เขียน<u>ตามใบเปิดโครงการ</u> วงเล็บชื่อภาษาอังกฤษ (ถ้ามี)
3. **วันที่/ช่วงเวลาที่จัดโครงการ** เขียนในรูปแบบ วว ดดดด ปปปป เช่น 1 เมษายน 2568 - 30 มิถุนายน 2568
4. **หน่วยงานที่รับผิดชอบ** (ถ้ามี) เช่น นักศึกษาแพทย์ชั้นปีที่ 2, ชมรมศิริราชโมชั่นพิกเจอร์
5. **อุปนายกที่ดูแลโครงการ/กิจกรรมนั้น** เช่น อุปนายกฝ่ายกิจกรรม, อุปนายกฝ่ายชมรม
6. คำว่า **“สโมสรนักศึกษาแพทย์ศิริราช”** เพราะถือเป็นองค์กรที่รับผิดชอบโครงการ
